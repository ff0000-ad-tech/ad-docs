<static> makeLinearGradient(args)

Returns a linear gradient between two given points as part of thie CanvasDrawer's context element

EXAMPLES
adding a 100x100 rectangle at 0, 0 with a linear gradient going from coordinate 0,0 to coordinate 0,100 fading from solid black to transparent black:
	adData.elements.myCanvasDrawer = new CanvasDrawer({
		id: 'myCanvasDrawer', 
		target: adData.elements.redAdContainer,
		css: {
			x: 0,
			y: 0,
			width: 300,
			height:	250
		},
		debug: false
	});
		var gradient = CanvasTexture.makeLinearGradient({
		target: adData.elements.myCanvasDrawer,
		xStart: 0,
		yStart: 0,
		xEnd: 0, 
		yEnd: 200, 
		colors: [	{stopVal: 0, color: 'rgb(0, 0, 0)'},
					{stopVal: 1, color: 'rgba(0, 0, 0, 0)'}
				]
	});
	new CanvasRect({
		target: adData.elements.myCanvasDrawer,
		id: 'myRect',
		params: {
			x: 0,
			y: 0,
			width: 100,
			height: 100
		},
		fill: gradient,
	});

<static> makePattern(args)

Returns a pattern given an image source and repeat style.

EXAMPLES:
Adding a circle with a radius of 400 at 0, 0 with a pattern from an image named 'tileJPG' which only repeats horizontally: 
	adData.elements.myCanvasDrawerPattern = new CanvasDrawer({
		id: 'myCanvasDrawer', 
		target: adData.elements.redAdContainer,
		css: {
			x: 0,
			y: 0,
			width: 300,
			height:	250
		},
		debug: false
	});
		var pattern = CanvasTexture.makePattern({
		target: adData.elements.myCanvasDrawerPattern,
		source: 'tileJPG', 
		repeat: CanvasTiling.X
	});
		new CanvasCircle({
		target: adData.elements.myCanvasDrawer,
		id: 'myCircle', 
		params: {
			x: 0,
			y: 0,
			radius: 400,
			startAngle: 0,
			endAngle: 360
		},
		fill: pattern
	});


Adding a circle with a radius of 400 at 0, 0 with a pattern from an the above canvas source which repeats horizontally and vertically: 
	adData.elements.myCanvasDrawerCanvasTiling = new CanvasDrawer({
		id: 'myCanvasDrawer', 
		target: adData.elements.redAdContainer,
		css: {
			x: 0,
			y: 0,
			width: 300,
			height:	250
		},
		debug: false
	});
		var pattern = CanvasTexture.makePattern({
		target: adData.elements.myCanvasDrawer,
		source: adData.elements.myCanvasDrawerPattern.canvas,
		repeat: CanvasTiling.COVER
	});
		new CanvasCircle({
		target: adData.elements.myCanvasDrawer,
		id: 'myCircle',
		params: {
			x: 0,
			y: 0,
			radius: 400,
			startAngle: 0,
			endAngle: 360
		},
		fill: pattern
	});

<static> makeRadialGradient(args)

Returns a radial gradient between two given points as part of thie CanvasDrawer's context element.

EXAMPLES
adding a circle with a radius of 400 at 0, 0 with a radial gradient starting from a radius of 50 to a radius of 200 and fading from solid red to transparent green: : 
	adData.elements.myCanvasDrawer = new CanvasDrawer({
		id: 'myCanvasDrawer', 
		target: adData.elements.redAdContainer,
		css: {
			x: 0,
			y: 0,
			width: 300,
			height:	250
		},
		debug: false
	});
		var gradient = CanvasTexture.makeRadialGradient({
		target: adData.elements.myCanvasDrawer,
		xInner: 0,
		yInner: 0,
		radiusInner: 50,
		xOuter: 0,
		yOuter: 0,
		radiusOuter: 200,
		colors: [ 	{stopVal: 0, color: 'rgb(255, 0, 0)'},
						{stopVal: 1, color: 'rgba(0, 255, 0, 0)'} 
					]
	});
	new CanvasCircle({
		target: adData.elements.myCanvasDrawer,
		id: 'myCircle',
		params: {
			x:0,
			y:0,
			radius: 400,
			startAngle: 0,
			endAngle: 360
			}, 
		fill: gradient
